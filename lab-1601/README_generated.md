## 📝 Документация проекта: «Проверка и обработка данных»

Этот проект содержит набор полезных утилит для работы с электронной почтой, сортировкой словарей и замером производительности функций.

---

### 🗂 Описание проекта

Проект включает три основные функции:

1. **validate_email** — проверка валидности адреса электронной почты с помощью регулярных выражений.
2. **sort_by_key** — сортировка списков словарей по заданному ключу.
3. **timer** — декоратор для измерения времени выполнения любой функции.

---

### 📌 Пример использования

#### ✅ Проверка валидности email:

```python
email = 'example@example.com'
print(validate_email(email))  # Вернет True, если email валиден
```

#### ⚙️ Сортировка словаря по ключу:

```python
data = [
    {'name': 'Иван', 'age': 25},
    {'name': 'Анна', 'age': 30},
    {'name': 'Сергей', 'age': 20}
]
sorted_data = sort_by_key(data, 'age')
for user in sorted_data:
    print(user['name'], user['age'])
```

Результат:
```
Сергей 20
Иван 25
Анна 30
```

#### 🕰 Измерение времени выполнения функции:

```python
@timer
def slow_function():
    time.sleep(2)
slow_function()  # Выведет время выполнения функции
```

### 💡 Основные функции

#### validate_email(email: str) → bool

Функция проверяет валидность email-адреса.

Параметры:
- `email` (строка) — email-адрес для проверки.

Возвращает:
- `True`, если email валидный, иначе `False`.

#### sort_by_key(data: List[Dict], key: str, reverse: bool = False) → List[Dict]

Сортировка словарей по заданному ключу.

Параметры:
- `data` (список словарей) — исходный список словарей.
- `key` (строка) — ключ, по которому нужно отсортировать.
- `reverse` (логический флаг, опционально) — сортируется ли список в обратном порядке (по умолчанию `False`).

Возвращает:
- Списки отсортированных словарей.

Исключения:
- `KeyError`: если указанный ключ отсутствует хотя бы в одном словаре.
- `TypeError`: если переданные параметры имеют неправильный тип.

#### timer(func: Callable) → Callable

Декоратор для измерения времени выполнения произвольной функции.

Параметры:
- `func` (функция) — любая функция, которую нужно обернуть.

Возвращает:
- Декорированную функцию с дополнительным функционалом замера времени выполнения.
